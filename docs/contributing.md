# Contributing to oneshot

Thank you for contributing to oneshot!

## Development Workflow

This project follows the **Agent Kernel** development workflow. All contributors (including AI agents) must follow the A-E workflow:

1. **Analyze** the request.
2. **Build** a project plan.
3. **Code** the implementation.
4. **Document** changes.
5. **Evaluate** against the [Definition of Done](definition-of-done.md).

See **[AGENTS.md](../AGENTS.md)** for detailed instructions.

## Code Style

This project follows the Agent Kernel code quality standards for Python. See:

- **[Python Definition of Done](system-prompts/languages/python/definition-of-done.md)** - Code quality standards
- **[Definition of Done](definition-of-done.md)** - Project-specific requirements

## Getting Started

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `pytest`
